# Iris ML Classification Model

A beginner-friendly machine learning project that classifies Iris flowers into their respective species using flower measurements.

This project was developed as part of my second-semester machine learning learning journey. It demonstrates the basic end-to-end workflow of a supervised classification problem, from loading data to training and evaluating a model.

---

## Project Overview

The Iris dataset contains flower measurements for three Iris species:

- Iris-setosa
- Iris-versicolor
- Iris-virginica

The objective is to predict the flower species using the following features:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

---

## Machine Learning Workflow

The project follows these steps:

1. Import required Python libraries.
2. Load and inspect the Iris dataset.
3. Identify input features and target labels.
4. Split the dataset into training and testing sets.
5. Train a Random Forest Classifier.
6. Make predictions on test data.
7. Evaluate the model using accuracy score.
8. Analyze feature importance.
